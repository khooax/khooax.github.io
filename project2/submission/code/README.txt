Description:
This project implements convolution, edge detection, Gaussian filtering, sharpening, hybrid images, and multiresolution blending.

Requirements: 
Install dependencies before running with
pip install numpy scipy matplotlib pillow scikit-image opencv-python

File Structure:
submission/
│
├─ inputs/                  <- Contain all input images used in code 
├─ code/
│   └─ main.py  
│   └─ align_image_code.py  
│   └─ README.txt           
├─ web/
│   └─ page.py         

How to Run:
Run the script directly with
python main.py
Each part prints progress in the terminal and shows figures via matplotlib.
